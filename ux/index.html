MY WORKS

Beta Engine

An engine made for a Year 1 game project.

The editor's interface and various functionalities are focused on building the game rapidly.

I developed some non-typical features here such as a robust animation editor, prefab tiler, and a dialogue editor.

Forge Engine

An engine made for a Year 2 & 3 game project.

The editor's interface and various functionalities are focused on future-proofing and extendibility.

I implemented many features with an API for other users such as C# scripting integration, flexible file I/O, and prefab technology.

Others
